# SpecLens 설정 가이드

## 파일 구조

```
speclens/
├── index.html        ← 브라우저에서 열 메인 파일
├── style.css         ← 스타일
├── app.js            ← 앱 로직
├── sheets-sync.js    ← Skill 1: 시트 연동
├── req-analyzer.js   ← Skill 2: AI 분석
└── Code.gs           ← Google Apps Script (아래 가이드 참고)
```

---

## Step 1 — Google Apps Script 설정

### 1-1. Apps Script 편집기 열기

1. 스프레드시트(`Next플랫폼기획 2팀`)를 열어요.
2. 상단 메뉴 → **확장 프로그램** → **Apps Script** 클릭

### 1-2. 코드 붙여넣기

1. 편집기에서 기존 코드를 모두 지우고
2. `Code.gs` 파일의 내용을 전체 복사해서 붙여넣어요.
3. 💾 **저장** (Ctrl+S / Cmd+S)

### 1-3. Anthropic API 키 등록

1. 편집기 왼쪽 메뉴 → **프로젝트 설정** (⚙️ 아이콘)
2. 스크롤 내려서 **스크립트 속성** → **속성 추가**
3. 아래와 같이 입력 후 저장:
   - 속성: `ANTHROPIC_API_KEY`
   - 값: `sk-ant-...` (Anthropic Console에서 발급한 키)

> API 키 발급: https://console.anthropic.com → API Keys

### 1-4. 웹 앱으로 배포

1. 편집기 오른쪽 상단 → **배포** → **새 배포**
2. 유형 선택: **웹 앱**
3. 설정:
   - 실행 계정: **나 (스프레드시트 소유자)**
   - 액세스 권한: **모든 사용자**
4. **배포** 클릭
5. 생성된 **웹 앱 URL** 복사해두기
   - `https://script.google.com/macros/s/.../exec` 형태

---

## Step 2 — SpecLens 실행

1. `index.html` 파일을 브라우저에서 열어요. (더블클릭)
2. 처음 실행 시 설정 화면이 나타나요.
3. Step 1-4에서 복사한 **웹 앱 URL**을 입력하고 **연결하고 시작하기** 클릭!

## 자주 묻는 질문

**Q. 팀원들도 사용할 수 있나요?**
네! `speclens` 폴더를 공유 드라이브나 팀 서버에 올려두면 팀원 모두 같은 URL로 접속해 사용할 수 있어요. 각자 Apps Script URL만 설정하면 돼요.

**Q. "저장 중 오류"가 나요.**
Apps Script 배포 시 **액세스 권한을 "모든 사용자"** 로 설정했는지 확인해주세요. 배포 설정을 바꾼 경우 **새 버전으로 재배포**가 필요해요.

**Q. AI 분석이 오래 걸려요.**
요구사항 수가 많을수록 오래 걸릴 수 있어요. Apps Script의 최대 실행 시간(6분)을 넘으면 오류가 날 수 있어요. 모듈별로 나눠서 분석하는 현재 구조가 가장 안전해요.

**Q. 코드를 수정했는데 반영이 안 돼요.**
Apps Script 코드를 수정하면 **재배포(새 버전 배포)**를 해야 반영돼요.
